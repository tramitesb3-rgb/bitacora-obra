Conectar Google Drive

Ingresá tu API Key y el ID de la carpeta de Drive donde se guardará la bitácora. Los datos se guardan como bitacora_obra.json.

Cómo obtener las credenciales:
1. Ir a console.cloud.google.com
2. Crear proyecto → Habilitar Google Drive API
3. Crear credenciales → API Key (para leer) y OAuth 2.0 Client ID (tipo Web, con http://localhost y la URL de este archivo en orígenes autorizados)
4. El ID de carpeta es la última parte de la URL de Drive al abrir la carpeta
